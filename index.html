  9. Arrays

    Array is a group of like-type variables
    that are referred by a common name,
    having continuous memory. Primitive values
    or objects can be stored in an array.
    It provides code optimization since we can
    sort data efficiently and also access it randomly.
    The only flaw is that we can have a
    fixed-size elements in an array.

    There are Two kinnds of arrays defined in Java:

    • Single Dimentional:Elements are stored in a single row.

    • Mutli Dimentional:Elements are stored as row and column.
